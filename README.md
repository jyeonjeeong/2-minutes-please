# ShutUp Chat 🤐

혼자 2분 넘게 말하면 마이크가 강제로 꺼지는 화상채팅

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000` 접속

## 핵심 규칙

| 규칙 | 설명 |
|------|------|
| 2분 제한 | 연속 발화 시간이 2분을 넘으면 마이크 강제 OFF |
| 10초 정적 | 10초 넘게 침묵하면 타이머 리셋 (새 턴) |
| 10초 이내 | 10초 안에 다시 말하면 타이머 이어서 계속 |
| 쿨다운 30초 | 셧다운 후 30초간 마이크 잠금 |

## 시각 효과

- **발화 중**: 비디오 테두리 초록색 + 타이머 표시
- **셧다운**: 비디오 흑백 처리 + 빨간 테두리 + 🤐 오버레이

## 기술 스택

- WebRTC (P2P 화상채팅)
- Web Audio API (음성 감지)
- Socket.io (시그널링 서버)
- Express (정적 파일 서빙)
